# ALU-8-bits
Unidade de Lógica e Aritmética para entrada e saída 8 bits.

Construída no [Digital](https://github.com/hneemann/digital)

[Vídeo demonstração](https://drive.google.com/file/d/1WFhapyYoE33MdLRWk5QlaW8zfDG4UkJv/view?usp=sharing)

Inclui o arquivo do circuito da ALU principal, assim como todos os componentes utilizados para constituir ela.

Operações:
- OP = 1: AC + N
- OP = 2: AC - N
- OP = 3: AC * N
- OP = 4: AC / N
- OP = 5: AC shift lógico esquerda
- OP = 6: AC shift lógico direita
- OP = 7: AC NAND N
- OP = 8: AC XOR N
